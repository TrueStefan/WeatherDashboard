Run:

1. While in WeatherDashboard\WeatherDashboardApi\WeatherDashboardApi do: dotnet run
2. While in WeatherDashboard\weather-dashboard-ui do: npm run dev
3. CTRL click the link

Ensure you have latest versions of .NET and Node.js
